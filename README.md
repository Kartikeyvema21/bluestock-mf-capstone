# Mutual Fund Analytics Platform

A comprehensive analytics system for mutual funds including ETL, performance metrics, risk analytics, and recommendation engine.

## Project Structure
- **data/**: Raw, processed, and SQLite database
- **notebooks/**: 5 Jupyter notebooks for EDA and analytics
- **scripts/**: Python modules for ETL, live NAV, metrics, recommendation
- **sql/**: Database schema and analytical queries
- **dashboard/**: Power BI dashboard file (.pbix)
- **reports/**: Final report PDF and presentation PPTX

## Setup
1. Create virtual environment: python -m venv venv
2. Activate: .\venv\Scripts\Activate.ps1
3. Install: pip install -r requirements.txt

## Usage
Run notebooks in order: 01 → 02 → 03 → 04 → 05
